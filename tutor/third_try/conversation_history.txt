**Installation Options:**

You can choose between two installation methods:

1. **CDN (Content Delivery Network):**
	* Add the following `<script>` tag in your HTML file:
```html
<script src="https://cdn.jsdelivr.net/npm/p5.quadrille.js@latest/dist/p5.quadrille.min.js"></script>
```
2. **npm (ESM) and Vite:**
	* Install the library using npm or yarn:
```
npm install p5-quadrille
```
	* Create a new project with Vite or another bundler, and import the library in your JavaScript file:
```javascript
import { createQuadrille } from 'p5-quadrille';
```

**Creating a Simple Quadrille:**

Once installed, you can start using the `createQuadrille` function to create a quadrille. Here's an example:

```javascript
function setup() {
  createQuadrille({
    cellLength: 20,
    width: 400,
    height: 400,
    origin: 'center'
  });
}

function draw() {
  background(220);
  drawQuadrille();
}
```

In this example, we're creating a quadrille with a cell length of 20 pixels, a width and height of 400 pixels each, and an origin set to the center. We then render the quadrille onto the canvas in the `draw` function.

That's it! You've successfully installed and used the `p5.quadrille.js` library to create a simple quadrille.
