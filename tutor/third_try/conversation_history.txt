 content: To use the `reflect()` method in your code, you can simply call it on an instance of the `Quadrille` class. Here's an example:

```js
let quadrille;

function setup() {
  createCanvas(4 * Quadrille.cellLength, 4 * Quadrille.cellLength);
  quadrille = createQuadrille(4, 4, 3, '🚀');
  quadrille.rand(4, '🐒');
}

function draw() {
  background('orange');
  drawQuadrille(quadrille);
}

function mouseClicked() {
  quadrille.reflect();  
}

function keyPressed() {
  quadrille.reflect();
}
```

In this example, the `reflect()` method is called when the user clicks the mouse or presses a key. This will reflect the quadrille across the horizontal axis.

Note that you need to create an instance of the `Quadrille` class and call the `reflect()` method on it. You can also chain multiple transformations together by calling them in sequence, like this:

```js
quadrille.rotate().reflect().transpose();
```

This will first rotate the quadrille 90 degrees clockwise, then reflect it horizontally, and finally transpose it.
