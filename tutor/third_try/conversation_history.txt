To install the **p5.quadrille.js** library, you can follow these steps:

**CDN (Content Delivery Network) Installation**

1. In your HTML file, add the following script tags:

```html
<!-- index.html -->
<!-- Load p5.js first (required by p5.quadrille.js, latest version) -->
<script src="https://cdn.jsdelivr.net/npm/p5/lib/p5.min.js"></script>
<!-- Load p5.quadrille.js (latest stable version) -->
<script src="https://cdn.jsdelivr.net/npm/p5.quadrille/dist/p5.quadrille.min.js"></script>
```

2. Open your HTML file in a browser or use a local development environment like VSCodium or Visual Studio Code with the Live Server extension.

**npm (Node Package Manager) Installation**

1. Run the following command in your terminal:

```bash
npm i p5 p5.quadrille
```

2. In your JavaScript file, import both libraries:

```javascript
// main.js
import p5 from 'p5'
import Quadrille from 'p5.quadrille'

const sketch = () => {
  // Your code here...
}

new p5(sketch);
```

That's it! You should now have the **p5.quadrille.js** library installed and ready to use in your projects.

To create a simple quadrille using the `p5.quadrille.js` library, you can use the `createQuadrille()` method. Here's an example:

```javascript
// Create a 10x10 quadrille with a random image in each cell
function setup() {
  let img = createImage(10, 10);
  for (let i = 0; i < 100; i++) {
    let x = floor(random(10));
    let y = floor(random(10));
    let c = color(random(255), random(255), random(255));
    img.set(x, y, c);
  }
  createQuadrille(10, img);
}

// Draw the quadrille
function draw() {
  background(0);
  noStroke();
  for (let row = 0; row < height; row++) {
    for (let col = 0; col < width; col++) {
      let val = getCellValue(col, row);
      fill(val);
      rect(col * 10, row * 10, 10, 10);
    }
  }
}
```

In this example, we create a 10x10 image and set the color of each cell randomly. Then, we use the `createQuadrille()` method to generate a quadrille with the random image in each cell.

The `draw()` function draws the quadrille onto the canvas by iterating over each cell and drawing it as a rectangle filled with its corresponding color value.

This is just a simple example of creating a quadrille. You can customize the creation process, draw styles, and visualizations to suit your needs.

Remember, the `p5.quadrille.js` library provides various algorithms for sorting, filtering, and transforming quadrille cells, which you can explore in more detail using the documentation.
