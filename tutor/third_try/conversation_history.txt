2. **Set the cell length**: For a simple quadrille, you can set the cell length to a fixed value. Let's say you want each cell to be 50 pixels wide:
   ```javascript
   Quadrille.cellLength = 50;
   ```

3. **Create the quadrille**: Using the `createQuadrille` method, generate a new 8x8 quadrille with a chessboard pattern:

4. **Set up the canvas**: In the `setup` function, create a canvas with the desired size (in this case, 8 times the cell length):
   ```javascript
   function setup() {
     createCanvas(8 * Quadrille.cellLength, 8 * Quadrille.cellLength);
     // ...
   }
   ```

5. **Draw the quadrille**: In the `draw` function, clear the background and draw the quadrille:
   ```javascript
   function draw() {
     background('beige');
     drawQuadrille(q);
   }
   ```

Here's the complete code example:
```javascript
import p5 from 'p5';
import Quadrille from 'p5.quadrille';

Quadrille.cellLength = 50;
let q;

function setup() {
  createCanvas(8 * Quadrille.cellLength, 8 * Quadrille.cellLength);
  q = createQuadrille(8, 8, ({ row, col }) => row === col || row + col === 7, '🌀');

function draw() {

  drawQuadrille(q);
}
```
This code creates a simple 8x8 quadrille with a chessboard pattern, where each cell is filled with the '🌀' symbol. You can modify the `createQuadrille` method to suit your specific needs.
